# Bongo Cat Desktop Overlay

Your desktop bongo cat :3

![Bongo Cat Demo](assets/demo.gif)

## Requirements
- Python 3.7+
- Windows (Recommended for global hooks)
- Administrator/Root privileges (Required by the `keyboard` library to intercept global keypresses).

## Installation
1. Clone the repository.
2. Install dependencies:
```bash
pip install -r requirements.txt
```


## Usage:
Run the script with administrator privileges:

```bash
  python main.py
```
